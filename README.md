# AI Economist Visual Lab

一个用于课程展示和期末答辩的多智能体经济体可视化实验平台。项目通过二维地图模拟多个 Agent 在资源环境中的采集、交易、建造、种树等行为，并对比不同政策模板对社会福利、公平性、生产力和生态恢复的影响。

## 项目特点

- 多智能体微观地图：在网格地图中实时观察 Agent、树木、矿石、市场和房屋。
- 政策对照实验：支持自由市场、AI Economist、累进税制、公平优先、效率优先等政策模板。
- 强化学习规划器：提供基于 `Q-learning` 的年度政策选择模式。
- 可视化指标面板：展示社会福利、平等度、生产力、生态指数、财富分布和行为统计。
- 数据导出：支持导出 CSV 和 JSON，便于后续整理实验报告。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务

```bash
npm run dev
```

启动后在浏览器中打开终端提示的本地地址，通常是：

```text
http://localhost:5173/
```


PowerShell 环境下同样可以使用：


```bash
npm run preview
```

## 使用说明

1. 在左侧“实验配置”中选择目标函数、政策模板、Agent 数量、资源密度、仿真年份和每年步数。
2. 点击“应用参数并重算”，系统会重新运行一次完整仿真。
3. 中间区域可以播放、暂停、单步查看经济体运行过程。
4. 点击地图上的 Agent 可以查看该个体的角色、财富、资源、房屋和行动状态。
5. 右侧“策略读数”展示当前政策下的核心指标、策略解释和强化学习信息。
6. 下方“行为分析”用于比较当前策略与基准策略，并观察趋势图、财富分布和事件流。

## 仿真模型概览

项目中的经济体由以下部分组成：

- Agent：拥有不同角色，包括采集者、建造者、交易者和林业者。
- 资源：地图上分布树木和矿石，树木存在成熟与树苗阶段。
- 市场：位于地图中心，交易型 Agent 会围绕市场产生经济活动。
- 房屋：Agent 消耗木材和石材后可以建造房屋，提升财富和生产力。
- 生态：种树、资源消耗和政策生态加成都影响生态指数。

每一步仿真都会更新 Agent 行为、资源状态、财富分布、国库、生产力、平等度、社会福利和生态指数。

## 政策模板

| 策略 | 说明 |
| --- | --- |
| 自由市场 | 税负较低，鼓励交易和建造，但财富分化更明显。 |
| AI Economist | 在公平、产出和生态之间做折中。 |
| 累进税制 | 提高再分配强度，适合作为对照组。 |
| 公平优先 | 强调缩小财富差距和生态恢复。 |
| 效率优先 | 强调产出增长、建造收益和市场活跃度。 |
| Q-learning 规划器 | 逐年观察状态，并从候选政策中选择预计回报更高的策略。 |

## Q-learning 逻辑

当政策选择为 `Q-learning 规划器` 时，系统会先训练一个轻量级策略选择器：

- 状态编码：由目标函数、年份阶段、平等度、生产力和生态状态组成。
- 动作空间：从固定政策模板中选择下一年度政策。
- 奖励函数：根据目标函数侧重社会福利、公平性、生产力、市场活跃度或生态指数。
- 输出结果：展示训练轮数、平均回报、学习率、折扣因子、探索率和年度策略轨迹。

## 答辩讲解建议

可以按下面的顺序介绍项目：

1. 先说明目标：用多智能体仿真观察不同政策对公平、效率和生态的影响。
2. 展示地图：说明 Agent 如何采集、交易、建房、种树。
3. 切换政策：对比固定策略与 `Q-learning` 规划器的表现差异。
4. 看指标：用社会福利、平等度、生产力、生态指数解释结果。
5. 导出数据：说明实验结果可以进一步用于报告分析。
